# Description of the produced dataset:

The produced dataset contains in each line an average of the selected variables agregated for each user and activity.There are following columns in the dataset:

* Activity - Contains the name of the activity performed
* User - Contains the name of the user

Other columns (see below) contain the averaged value of the measurments for a particular user and activity. For interpretation of the name of the variables, please look into the original data code book (included in https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip).
